# Wave Forecast Prediction Model

Wave Forecast Prediction Model is a group machine learning project focused on predicting ocean wave conditions using environmental data from a Kaggle dataset. The project involved data cleaning, relational dataset merging, feature engineering, model training, and performance evaluation across multiple machine learning models.

## Project Type

Team Project / Machine Learning Project

## My Role

Co-Creator

## Features

- Built a machine learning pipeline in Python using Jupyter Notebook
- Used a Kaggle dataset containing environmental and ocean wave-related data
- Merged and cleaned multiple relational datasets using primary and foreign keys
- Created a unified dataset for model training and evaluation
- Performed feature engineering, including one-hot encoding for categorical variables
- Transformed time-based features to improve model performance
- Trained and evaluated 10+ machine learning models
- Compared models using performance metrics such as R² and RMSE
- Improved data quality by removing redundant and non-informative features

## Models Used

- Linear Regression
- Random Forest
- Gradient Boosting
- Neural Networks
- Additional machine learning models tested during experimentation

## Tech Stack

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Machine Learning
- Data Cleaning
- Feature Engineering

## Purpose

The goal of this project was to predict ocean wave conditions from environmental data and compare different machine learning approaches to determine which models performed best. The project strengthened skills in data preprocessing, feature engineering, model evaluation, and applied machine learning.

## My Contributions

As a co-creator, I contributed to the machine learning pipeline, data cleaning, dataset preparation, feature engineering, model testing, and performance evaluation. I also collaborated with my team to compare model results and identify the best-performing approaches.

## Results

The project evaluated multiple machine learning models using R² and RMSE to compare prediction performance. The team used these metrics to identify stronger model approaches and improve the quality of the final dataset.
